難易度設定

置換基が2種類以上ある場合の命名法

全体の流れ

  1. 主鎖（最長の炭素鎖）または環を決める
  2. 置換基に位置番号をつける
  3. 決まった順番（アルファベット順）で名前を書く

位置番号の決め方

(a) 位置番号の組み合わせは最小になる方を選ぶ

  • 主鎖には両端があり、番号のつけ方は複数候補がある
  • 各候補の位置番号を順に並べ、先頭から順に比較し、最初に違いが出たところで判定

5-ブロモ-2-クロロ-2-フルオロ-1-ヨード-4-メチルペンタン

  • 左から：{1,2,2,4,5}／右から：{1,2,4,4,5}
  • 1〜2個目は同点、3個目で 2 < 4 → 「左から」が正しい

(b) 環の場合も考え方は同じ

  • 環には時計回り・反時計回りの2方向があり、(a)と同様に両方の位置番号の組み合わせを作って比較する
  • 置換基が1つだけの場合、位置番号は不要（例：メチルシクロヘキサン）

(c) (a)(b)で比較しても同点なら、アルファベット順で先の置換基に小さい番号

bromo < chloro < ethyl < fluoro < iodo < methyl

（英語のつづり順。五十音順ではない）

（コツ）判断に悩んだら

(d) 環状構造で、1つの炭素に置換基が2つある「二置換炭素」があれば、そこが1番の最有力候補

  • 二置換炭素を1番にすると、他のどの位置から始めるより位置番号の組み合わせが小さくなりやすい
  • 該当する炭素があれば、まずそこを1番候補として検討する

名前を書く順番

  • 位置番号の大小や、(c)のような同点判定の有無にかかわらず、置換基名は常にアルファベット順に並べて書く

具体例

例1：基本パターン

✅ 2-クロロ-3-メチルペンタン
❌ 4-クロロ-3-メチルペンタン

  • 炭素5個の鎖にクロロ・メチル。左から{2,3}、右から{3,4}
  • 1個目で 2<3 → 左からが正解
例2：2個目まで比べて決まる

✅ 6-ブロモ-2-クロロ-3-メチルヘプタン
❌ 2-ブロモ-6-クロロ-5-メチルヘプタン

  • 炭素7個の鎖にブロモ・クロロ・メチル。左から{2,3,6}、右から{2,5,6}
  • 1個目は同点（2）、2個目で 3<5 → 左からが正解
  • 「1個目が同じだから以降は不問」は誤り
例3：環＋真の同点（アルファベット順で決着）

✅ 1-エチル-2-フルオロシクロヘキサン
❌ 2-エチル-1-フルオロシクロヘキサン

  • 炭素6個の環にエチル・フルオロが隣接。両方向とも{1,2}で完全に同点
  • ethyl(e) < fluoro(f) → エチルに小さい番号1を与える
例4：二置換炭素が1位の最有力候補

✅ 2,6-ジブロモ-1-ヨード-1,3-ジメチルシクロヘキサン

  • 炭素6個の環にブロモ×2・ヨード×1・メチル×2
  • ヨードとメチルが同じ炭素につく箇所を1位とする
  • あとは時計回り・反時計回りで位置番号が小さくなる方を選ぶ
